# Smart Guardian - Unified Security Dashboard

Advanced safety and monitoring solution built with Flutter, designed to provide comprehensive protection and intuitive monitoring.

## Core Features
- **Real-time Monitoring**: Integrated intelligence for continuous safety oversight.
- **Smart Alerts**: Proactive notification system for critical events.
- **Modern UI**: Clean, professional interface optimized for both mobile and web.

## Tech Stack
- **Framework**: Flutter (Dart)
- **Platforms**: Android, iOS, Web
- **APIs**: Securely configured Google Maps integration.

## Getting Started
To get started with the project, ensure you have the Flutter SDK installed on your system.
1. Use `flutter pub get` to install dependencies.
2. Run with `flutter run`.
